# 🌽 Portal do Milho - Blog de Cultivo e Manejo

O **Portal do Milho** é um site responsivo estilo blog, focado em trazer informações técnicas, artigos e ferramentas interativas para produtores agrícolas, estudantes do agro e entusiastas que buscam otimizar o manejo da cultura do milho, desde o plantio até à colheita.

O projeto foi estruturado seguindo as boas práticas de desenvolvimento web, separando as responsabilidades de marcação (HTML), estilo (CSS) e lógica (JavaScript) em pastas organizadas.

---

## 🚀 Funcionalidades

* **Design Responsivo:** O layout adapta-se automaticamente a ecrãs de computadores, tablets e telemóveis.
* **Artigos Técnicos:** Seções prontas para conteúdos sobre sementes, adubação (NPK), controle de pragas (lagarta-do-cartucho e cigarrinha) e manejo hídrico.
* **Simulador de Plantio (Interativo):** Uma ferramenta em JavaScript que calcula a estimativa de sementes necessárias com base nos hectares digitados pelo usuário.
